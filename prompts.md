# Prompt Notes

## Character continuity core block
- Same red-haired freckled boy from the reference image
- Keep facial structure and eye shape consistent
- Keep hair color and freckles consistent

## Camera motion block examples
- slow push-in
- smooth orbit around subject
- handheld follow shot
- crane up reveal

## Avoidance block
- Do not start with the original close-up scene
- Avoid identical framing and lighting from reference
